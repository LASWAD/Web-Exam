المهارات

. تصميم واجهات المستخدم (Frontend Development):. “ تصميم صفحات ويب متجاوبة باستخدام HTML5, CSS3. إنشاء مواقع تفاعلية ومرئية باستخدام JavaScript ومكتبة jQuery. تطوير مواقع جذابة وقابلة للتخصيص باستخدام Bootstrap ونظام الشبكي (Grid System). “

البرمجة وتطوير التطبيقات (Backend Development): برمجة التطبيقات باستخدام لغة C# وتطبيق البرمجة الكائنية (OOP). تطوير تطبيقات ويب باستخدام ASP.NET Core MVC لتطبيق نموذج التصميم (Model/View/Controller). إنشاء تطبيقات ويب متقدمة باستخدام Blazor لإنشاء تطبيقات صفحة واحدة (SPA). اعتماد تقنيات حديثة مثل Repository Pattern وDependency Injection لضمان جودة الكود وقابليته للصيانة.

. إدارة قواعد البيانات (Database Management): تصميم قواعد بيانات باستخدام Entity Framework Core بطريقة (Code First). إنشاء الجداول والعلاقات وإدارة الهجرة (Migrations). دعم قواعد بيانات متنوعة مثل SQL Server, MySQL, PostgreSQL, SQLite.

. الأمن وإدارة المستخدمين1.: : استخدام ASP.NET Core Identity لإنشاء حسابات المستخدمين وإدارة الصلاحيات والأدوار. تأمين التطبيقات ضد الهجمات ومحاولات الاختراق. 2. أدوات وتقنيات متقدمة: استخدام إطار العمل AutoMapper لتحويل البيانات بين الطبقات المختلفة. برمجة الوظائف غير المتزامنة (Asynchronous Programming) لتحسين الأداء. إرسال الرسائل الإلكترونية برمجياً باستخدام خدمات البريد المختلفة. 3. نشر وإدارة المشاريع: رفع التطبيقات وقواعد البيانات على شبكة الإنترنت. التعامل مع شركات الاستضافة وإعداد أسماء النطاقات ولوحات التحكم. إدارة مشاريع متكاملة باستخدام لوحات تحكم احترافية. .4. مشاريع عملية: إنشاء تطبيقات ومواقع متكاملة تشمل جميع التقنيات التي تم تدريسها. تطبيق تحليل وتصميم الأنظمة مثل مواقع التواصل الاجتماعي ومنظومات إدارة المستخدمين.
